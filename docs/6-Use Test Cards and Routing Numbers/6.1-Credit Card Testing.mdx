# Credit Card Testing

Any valid credit card number can be used for testing. In this example, we will be using the test data
available in [Developer Studio â€“ CardPointe Integrated Payments](https://developer.fiserv.com/product/CardPointe/docs/?path=docs/documentation/CardPointeGatewayDeveloperGuides.md&branch=main#tab-uat_test_card_numbers_for_specific_responses). Scroll down until you reach the **UAT Test Card Data**. You will be using the **UAT Test Card Numbers for Specific Responses** tab to test for
multiple scenarios

:::warning[Testing]
Agencies are expected to test both positive and negative scenarios.
:::

<img src='/img/6/1.png' />

Type the **test card number** and **expiration information** into the necessary fields. Use any date in the
future for expiration date. The other fields can be filled with random information.
