# Validating API Transactions

Your SnapPay account includes sections for validating API Transactions. You will be able to view your request and response details. We will show you where to validate transactions below:

Log in to your [SnapPay Cert Account](https://snappaydirect-cert.fiserv.com/)
<img class="bordered shadow" src='/img/11/1.png' style={{width: 500}} />

<br /><br />

From the main screen toolbar, hover over **Inquiries** and click **Request Response**
<img class="bordered shadow" src='/img/11/2.png' />

<br /><br />

In the **Request Response** page, you will be able to see all data related to the API calls you make to SnapPay. If you click the **Transactions** dropdown, you can narrow down the type of transaction you want to validate.
<img class="bordered shadow" src='/img/11/3.png'  style={{width: 400}} />

<br /><br />

We will use **Interop** as an example.
Once we Select **Interop**, we can see all the request and response details related to API calls (in and out).
<img class="bordered shadow" src='/img/11/4.png' />


<br /><br />

To dig deeper into a specific request, click **View**. This will bring up the **Request Response Contents**.
<img class="bordered" src='/img/11/5.png'  style={{width: 400}} />
